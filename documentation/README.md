# Documentation

- Blockstack去中心化应用的准则
    - 用户控制他们自己的数据
        - 去中心化应用并不存储用户数据的副本
    - 用户自己管理自己的身份
        - 身份是唯一的（公私钥体系）并且不会被用户拿走
        - 用户可以拥有多个身份
    - 用户可以自由选择客户端
        - 数据与身份是基于应用独立的
        - 在一个应用中产生的数据应该可以被另一个应用访问。
